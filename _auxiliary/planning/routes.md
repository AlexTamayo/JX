RESTful Routes


| BREAD VERB            | HTTP Method | Endpoint          |
| :-------------------- | :---------: | :---------------- |
| Browse items          |     GET     | /items            |
| Read individual item  |     GET     | /items/:id        |
| Favourite item (Edit) |     POST    | /users/:id        |
| Add item              |     POST    | /items            |
| Delete / Sold         |     POST    | /items/:id/delete |
